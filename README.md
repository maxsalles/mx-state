**MX States** is a state management library for JavaScript apps.

## Prerequisites

You will need [Node.js](https://nodejs.org/) **version 12+**.

## Instalation

npm:

```
npm install @mx-js/states
```

yarn:

```
yarn add @mx-js/states
```
